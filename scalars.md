# {title}

## Implicit scalars

```
string: Hello World
integer: 42
float: 3.14
boolean: true
```

## Explicit scalars

```
string: !!str Hello World
integer: !!int 42
float: !!float 3.14
boolean: !!bool true
```

## Multi-line scalars

```
multiligne: |
  Ceci est un scalar multi-lignes
  Il peut contenir plusieurs lignes
```
