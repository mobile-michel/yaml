Data types

  • scalars: string, numbers, booleans

  • sequences: list of values

  • maps: key-value pairs

Scalars example

  • name: John Doe {name}

  • age: 30 {age}

  • employed: true {employed}

  • e-mail: john.doe@example.com {email}

Sequence example

List of hobbies:

  • hiking
  • photography
  • cooking
[sequence]

sequence.html file:

<div ="sequence">
    <p>List of hobbies:</p>
    <ul :for="item in hobbies">
        <li>{item}</li>
    </ul>
</div>

Map example

  • street: Main St. {address.street}

  • city: Anytown {address.city}

  • state: CA {address.state}
